# WordPress Troubleshooter 🔧

A diagnostic tool for troubleshooting common WordPress and WooCommerce issues.

## 🔗 Live Demo
[View Live Tool](https://akshayjhorarr.github.io/wordpress-troubleshooter)

## 💡 About This Project
Built to simulate real-world WordPress support workflows.
This tool helps users diagnose and resolve common issues
through a step-by-step guided flow.

## 🛠️ Features
- 6 issue categories (WooCommerce, Login, REST API, Migration, Performance, WSOD)
- Step-by-step diagnostic guides with root cause analysis
- Severity indicators (Low / Medium / High)
- Search by issue name or tag
- Estimated resolution time for each issue

## 📂 Categories Covered
- 🖥️ White Screen of Death (WSOD)
- 🛒 WooCommerce Checkout & Products
- 🔐 Login & Admin Access Issues
- ⚙️ REST API & Integrations
- 📦 Site Migration & Hosting
- ⚡ Performance & Speed

## 🧰 Built With
- HTML, CSS, JavaScript
- No frameworks, no dependencies
- Data-driven architecture
